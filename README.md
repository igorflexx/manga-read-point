# Manga Read Point

Telegram-бот для поиска манги на ReManga, отслеживания новых глав и сохранения точки чтения для каждой подписки.

## Основные файлы

- `main.py` - точка входа, которая запускает приложение бота.
- `mangabot/bot.py` - основная логика команд, меню, уведомлений и проверки обновлений.
- `mangabot/remanga.py` - работает с ReManga: поиск тайтлов и получение данных по главам.
- `mangabot/database.py` - хранит подписки, прогресс чтения и служебные данные в базе.
- `mangabot/models.py` - описывает структуры данных, с которыми работает бот.
- `mangabot/keyboards.py` - собирает клавиатуры для списка манги, поиска и действий с тайтлом.
- `mangabot/states.py` - хранит состояния пользовательских сценариев.
- `mangabot/config.py` - загружает токен и настройки проекта.
